# ♟ Chess AI Android App

Android application with Neural Network-based Chess AI.

## Features
- 🧠 Neural Network: 902→1024→768→512→[64+64] (2.2M params)
- 🔬 3-Phase Training: Imitation → REINFORCE → Self-Play
- 📱 Real-time board visualization
- 💾 Export/Import model weights
